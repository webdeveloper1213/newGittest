So far the proces sof git is extremely simple

  • In command line type cd documents or whereever you want to keep your git project folder. In my case it is in documents.
  • This is the first step in git. git clone "path of the folder." "https://github.com/webdeveloper1213/newGittest.git" in my case.
  • list the files using ls command . Then type cd newGittest which is the name of the repository/project.
  • Go to brackets and open this folder in your projects.
  • git status is the second command to use
  • git add index.html . check the status by typing git status
  • git commit -m "Added index.html"
  • git push to finally push the index.html into the github.

Now everytime you make some changes in the html file, you have to repeat these commands.

  1. git status
  2. git add -A
  3. git commit -m "comment here..."
  4. git push
